# 临床决策支持案例

## 案例：2型糖尿病患者的综合管理

### 患者基本信息

| 项目 | 内容 |
|------|------|
| 性别 | 男 |
| 年龄 | 52岁 |
| 主诉 | 口渴、多饮、多尿3个月，体重下降5kg |
| 既往史 | 高血压病史5年 |
| 家族史 | 父亲患2型糖尿病 |

### 临床表现

- 典型"三多一少"症状：多饮、多尿、多食、体重下降
- 无明显视力模糊
- 无肢体麻木

### 辅助检查

| 检查项目 | 结果 | 参考范围 | 判读 |
|----------|------|----------|------|
| 空腹血糖 | 9.8 mmol/L | 3.9-6.1 | ↑ 升高 |
| 餐后2h血糖 | 16.2 mmol/L | < 7.8 | ↑ 明显升高 |
| 糖化血红蛋白 | 8.5% | < 6.5 | ↑ 升高 |
| 尿微量白蛋白 | 18 mg/24h | < 30 | 正常 |
| 肌酐 | 78 μmol/L | 44-133 | 正常 |
| 总胆固醇 | 5.8 mmol/L | < 5.2 | ↑ 偏高 |
| BMI | 26.5 kg/m² | 18.5-23.9 | ↑ 超重 |

---

### 知识图谱推理过程

基于知识图谱进行临床决策推理：

```
患者症状: 多饮、多尿、多食、体重下降
    ↓ HAS_SYMPTOM (反向查询)
匹配疾病: 2型糖尿病 (ICD-10: E11)
    ↓ 确认检查
诊断依据: 空腹血糖 ↑, 糖化血红蛋白 ↑
    ↓ TREATED_BY
推荐药物: 二甲双胍 (一线), 恩格列净 (二线/合并肾保护)
    ↓ COMORBID_WITH
合并症关注: 高血压 (需联合管理)
    ↓ DIAGNOSED_BY
随访监测: 尿微量白蛋白 (肾病筛查), 糖化血红蛋白 (血糖控制)
```

### 诊断

1. **2型糖尿病**（依据：典型症状 + 空腹血糖 ≥ 7.0 mmol/L + HbA1c ≥ 6.5%）
2. **高血压病**（既往诊断）
3. **超重**（BMI 26.5）

---

### 治疗方案

#### 1. 药物治疗

| 药物 | 用法 | 作用 | 证据等级 |
|------|------|------|----------|
| 二甲双胍 | 500mg bid，逐渐加量至1000mg bid | 一线降糖，改善胰岛素敏感性 | A |
| 缬沙坦 | 80mg qd | 降压 + 肾脏保护 | A |
| 阿托伐他汀 | 20mg qn | 调脂、心血管保护 | A |

**备选方案（如血糖控制不佳）：**
- 加用恩格列净 10mg qd（兼具心肾保护，推荐用于合并心血管风险的T2DM患者）

#### 2. 生活方式干预

- **饮食管理**：低糖低脂饮食，控制总热量摄入
- **运动处方**：每周 ≥ 150 分钟中等强度有氧运动
- **体重管理**：目标减重 5-10%
- **戒烟限酒**

#### 3. 监测计划

| 项目 | 频率 | 目标 |
|------|------|------|
| 空腹/餐后血糖 | 每日（自我监测） | 空腹 < 7.0, 餐后 < 10.0 |
| 糖化血红蛋白 | 每3个月 | < 7.0% |
| 尿微量白蛋白 | 每年 | < 30 mg/24h |
| 肾功能 | 每年 | 正常范围 |
| 眼底检查 | 每年 | 无视网膜病变 |
| 足部检查 | 每次就诊 | 无溃疡/感染 |
| 血脂 | 每6个月 | LDL < 2.6 mmol/L |
